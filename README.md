# Nesti Website v6

Това е v6 версия на лендинг сайта за Nesti.

## Какво е сменено
- Цялостна визуална подредба, близка до одобрения mockup.
- Реалната начална страница на приложението е показана като основен app preview.
- „Създай списък“ отваря отделен примерен checklist modal.
- „Експортирай в PDF“ отваря отделен preview modal за примерния отчет.
- Store бутоните имат демо поведение и могат да се заменят с реални линкове.
- Всички кликаеми елементи имат hover / press ефект.
- Добавени са нови визуални assets за семейството и „Багаж с Nesti“.

## Файлове
- `index.html` — основна страница
- `styles.css` — стилове
- `script.js` — интеракции и modal логика
- `privacy.html`, `terms.html`, `support.html` — допълнителни страници

## Забележка
Ако искаш, следващата стъпка е v6.1:
- реални App Store / Google Play линкове
- отделна страница за PDF вместо modal
- отделна страница за „Багаж с Nesti“ вместо modal
- още реални app screenshots при наличност


## v6.2 — Development periods compact patch
- Компактна секция вместо огромен блок.
- Телефонният preview показва динамичната карта между шестте модула и „Багаж с Nesti“.
- Картата е ясно описана като временна — появява се само около релевантен възрастов прозорец.
- Оригиналният заключен Home остава референция извън тези периоди.
