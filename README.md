# Nesti Website v1

Реален responsive статичен сайт за Nesti.

## Файлове
- index.html
- styles.css
- script.js
- support.html
- privacy.html
- terms.html
- assets/

## Стартиране локално
Отвори `index.html` в браузър.

## За публикуване
Може да бъде качен директно в GitHub Pages, Netlify, Vercel или друг static hosting.

## Забележки
- App Store / Google Play бутоните са placeholders до реалните store links.
- hello@nesti.app е placeholder, ако финалният support email е различен.
- Pricing секцията използва текущия тестов модел Free / Plus / Premium.
- Legal страниците са placeholders до финален юридически review.
