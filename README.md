# Nesti Website v2

Подобрена версия, максимално близка до одобрения landing-page дизайн.

Основни промени спрямо v1:
- премахнати emoji пингвини;
- използвано е реалното Nesti лого от одобрения дизайн;
- използвани са реалните Nesti визуални елементи и икони;
- мобилните изображения вече не се crop-ват с `object-fit: cover`;
- на mobile изображенията се показват с естественото им съотношение;
- текстът е HTML, а не изрязан текст вътре в големи screenshot crop-ове;
- layout-ът следва по-близо оригиналния landing design;
- Free / Plus / Premium остава текущият тестов модел.

За обновяване на GitHub Pages:
Замени файловете от v1 със съдържанието на този ZIP и замени/добави файловете в `assets/`.
